Frequently Asked Questions

What is Nextride?
Nextride is a software that displays information about the departure times of streetcars. There is now a new version, Nextride2, which can also display other information. Nextride2 also has the ability to receive and display images. You can find more information in the repository of Nextride2
What is Nextride2-infobeamer
Nextride2-infobeamer short N2i, is a extension for Nextride2. N2i in the first place allows you to display many images and ressources after each other on the screens running Nextride2. You can either input a xml schedule in frab format or specify just a single image url. For every url N2i itteratively sents it to the Nextride2 screens. N2i also provide a CMS for user uploads. This is mostly used for events, where visitors should have the possibility to run ads or promote their spaces/projects.
What filetypes can I upload?
Curently you can only upload images (png, jpg, jpeg, gif). This restrictions comes from Nextride2 itself. Whenever Netride is able to play file like mp4, N2i can react on this.
